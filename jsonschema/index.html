JSon schemas

If you found this you are using the undocumented api from adapter.php. Documentation to come, but here are the json schemas.

http://skywatch.code-monkey.se/jsonschema/citiesschema.json
Schema for when multiple cities are returned.
http://skywatch.code-monkey.se/jsonschema/forecastschema.json
Schema for a single city hit with forecasts.
http://skywatch.code-monkey.se/jsonschema/errorschema.json
Schema for error returns.
